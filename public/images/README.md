## Image Guidelines
- Use high-quality images (JPG or PNG format)
- Optimize images for web to ensure fast loading times
- Ensure images are properly licensed for commercial use
- All images should be relevant to solar energy and home installation

## Fallback Behavior
If images are not found, the application will:
- Hero and About sections: Fall back to Unsplash placeholder images
- Service cards: Display without images (icons will still show)
- Testimonials: Display initial-based avatars if profile images are missing
